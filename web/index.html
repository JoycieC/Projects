US English Version Alpha v0.1.7

A modification of the English patch by Arks-Layer to alter various NPC, Enemy, Item and Area names to match their official names based on released merchandise, in-game credits and so forth. This is applied on-top of an existing installation of the Arks-Layer patch using a .bps patch, no content from the Arks-Layer patch is included. This patch is for testing purposes only and is not intended for widespread rlease. Instructions: Enable 'Manual patch' Click here: Projects/web/img/deumantweaker.png Then here: Whils using the PSO2 Tweaker, enable manual patching, this will stop the PSO2 Tweaker from deleting your patch.tar file and replacing it with the Arks-Layer version every time it boots up.:
