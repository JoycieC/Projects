PHANTASY STAR ONLINE 2

Deuman Patch

PSO2 Deuman Patch (Private repo, with public release) A modification of the English patch by Arks-Layer to alter various NPC, Enemy, Item and Area names to match their official names based on released merchandise, in-game credits and so forth.

This is applied on-top of an existing installation of the Arks-Layer patch using a .bps patch, no content from the Arks-Layer patch is included.

This patch is for testing purposes only and is not intended for widespread rlease.

Instructions:

Whilst using the PSO2 Tweaker, enable manual patching, this will stop the PSO2 Tweaker from deleting your patch.tar file and replacing it with the Arks-Layer version every time it boots up.
We use Beat (Flips can also be used) to patch the PSO2 translation using a .bps patch file, this allows us to offer our changes whilst not using any of the source content. If we can find a better way to provide the patches we'll look into it. If you're using the PS Vita version, the process is the same as below except you must place your patched cpk into the appropriate RePatch folder on your PS Vita. To start, you must use one of the following:


Download Beat here

Download Flips here



PSO2 Tweaker:



Enable Manual Patch:



Navigate to your PSO2 installation directory, generally this is similar to PHANTASYSTARONLINE2\pso2_bin\patches. Once you have located this, open beata and apply bps patch to English Patch to patch.tar:




Once you have done this, ensure that the patched file is named patch.tar and place this in the 'patches' folder.
Start PSO2 via the PSO2 Tweaker and you should now be running the patch.
