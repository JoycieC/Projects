FAQ


Q) What is this?
A) A patch intended to alter various NPC, Enemy, Item and Area names to match their official names based on released merchandise, in-game credits etc.

Q) When was this project started?
A) 18/07/2017 - Yes, 2017. The project was used internally by us for a long time until we lost interest in the game, we've since continued work starting from scratch and hope to continue as time goes on.

Here are some screenshots taken back in 2017:

Change to Wopal - Instances of Vopar being changed to Wopal, its correct name.
Wii U Port - An attempt at breathing new life into the late Wii U, also Xbox support in 2017!
Obligatory Star Fox reference - Note the Client Version, this was a long time ago now!

Q) How do I apply this?
A) Check below for instructions.

Q) Does this use any content from the open source patch by Arks-Layer?
A) No, this only changes things in the existing Arks-Layer patch, this is not an English patch on its own and requires the Arks-Layer patch to function.

Q) Who made this?
A) JoycieC and his team are behind the Deuman patch.

Q) Who's in the team? Can I meet them? Do you play PSO2?
A) Our team consists of a small circle of friends who work on projects like this for fun. For reasons of their own, the others do not wish to disclose their involvement.

Q) How can I revert back to the original patch?
A) Delete the patch and re-enable automatic updates in the Tweaker.

Q) Your website isn't very good...
A) Because none of us are Web Devs, we're beginner at best. Let us know if you can provide an alternative!

Q) Is everything really correct?
A) Based on official merch and the end credits, we've known for years that Lisa ir Risa, Jean is Jan, Claris Claes is Klariskrays, Klotho is Chroto, and so forth. Franca's name is spelled in plain (PSO2 style) English at the top of her cafe, yet for some reason she is still 'Franka' in the Arks-Layer patch. One of the biggest arguments for some of these names is "SEGA doesn't know English!!!!" and "But the space satelites are the basis for the names!". whilst this may be true in some cases and holds merit, one must also consider that Phantasy Star in and of itself has a history of bastardising many names, it's part of the franchise and a very important thing to consider. It's done on purpose, not always due to bad translation. "But what about the formal wear in PSU!", in an instance like that, it seems someone made a typo and it never got corrected to FOmarl as it should have in Q&A, that's all.

Q) Have you guys tried working with Arks-Layer instead?
A) Not directly, but in the past we have expressed the desire for optional name changes to their official ones, (Klotho vs Chroto, Vopar vs Wopal), however this was never met positively, which is why this project was started.

Instructions

Whilst using the PSO2 Tweaker, enable manual patching, this will stop the PSO2 Tweaker from deleting your patch.tar file and replacing it with the Arks-Layer version every time it boots up.

We use Flips (Beat can also be used) to patch the PSO2 translation using a .bps patch file, this allows us to offer our changes whilst not using any of the source content. This is essentially storing the difference in data between the patches, and applying the difference from ours to the original.

If we can find a better way to provide the patches we'll look into it.
If you're using the PS Vita version, the process is the same as below except you must place your patched cpk into the appropriate RePatch folder on your PS Vita. More detailed instructions will be available at a later date.

Download Flips here


Open the PSO2 Tweaker and select 'Tweaker Settings'



Enable checking for updates manually



Navigate to your PSO2 installation directory, generally this is similar to:

PHANTASYSTARONLINE2\pso2_bin\patches

Run flips.exe that you downloaded earlier and select 'Apply Patch'



Select the Deuman Patch of your choice



Select the original patch (patch.tar)



Select to save your new patch over the old patch



It will ask to confirm replacing the patch, select 'Yes'




Once you have done this, start PSO2 via the PSO2 Tweaker and you should now be running the Deuman Patch. Please report any bugs via the git repo. Please note that this patch can and will break when new updates are provided to PSO2, as such it is recommded to always check for the latest version before using this patch. To revert back to the original patch, just re-enable automatic updates in the Tweaker Settings and it'll do the rest.
